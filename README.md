# Infotecs autotest 

https://github.com/aazat3/Infotecs_test

Консольная асинхронная программа для тестирования доступности HTTP-серверов и измерения скорости их ответа.
Программа отправляет указанное количество запросов к каждому хосту и выводит статистику по результатам выполнения.

## Возможности
* Тестирование одного или нескольких хостов
* Поддержка чтения списка хостов из файла
* Подсчёт:
    * Success — успешные ответы (100–399)
    * Failed — ответы с ошибкой (400–599)
    * Errors — ошибки соединения (сервер недоступен)
    * Min — минимальное время ответа
    * Max — максимальное время ответа
    * Avg — среднее время ответа
* Асинхронное выполнение запросов (ускоренная работа)
* Валидация входных параметров
* Возможность сохранить результат в файл
* Проверка корректности входных параметров

## Параметры
| Ключ | Описание 
|---------|-------------|
| **-H, --hosts** | Список хостов через запятую (формат: https://example.com)
| **-F, --file** | Файл со списком хостов (по одному на строку) 
| **-C, --count** | Количество запросов к каждому хосту (по умолчанию 1)
| **-O, --output** | Файл для сохранения результата (если не указан — вывод осуществляется в консоль, иначе только в файл)

⚠ Одновременно можно использовать только один из ключей -H или -F.
Посмотреть информацияю о параметрах можно с помощью 
```bash
python bench.py -help
```

## Обработка ошибок
* Проверяется формат URL
* Проверяется корректность значения count
* Ошибки чтения/записи файлов обрабатываются через try/except
* Ошибки соединения учитываются отдельно в поле Errors
